# 横向研发文档

*   接入指引：应用可通过调用 CIAM 提供的标准化 [[API|CIAM API 文档]] 实现注册、登录、密码管理、多因素认证、账号自助服务（如资料修改、安全设置、注销等）等核心能力；接入前需在 IDaaS 控制台完成应用创建、配置回调地址、获取 Client ID/Secret，并完成 OAuth 2.0 / OIDC 协议对接；推荐使用阿里云提供的 [[SDK|CIAM SDK 支持列表]] 加速集成。

*   产品对接方案细节：支持 OAuth 2.0 Authorization Code Flow、Implicit Flow、Resource Owner Password Credentials（仅限可信内部应用）、Client Credentials（用于服务间调用）等多种授权模式；提供标准 OIDC Discovery Endpoint 及 UserInfo Endpoint；支持自定义登录页、品牌化 UI 组件（Logo、主题色、文案）、短信/邮件模板本地化配置；私有化部署场景下支持与企业现有 LDAP/AD、数据库、短信网关、邮件系统等通过 [[对接适配器|CIAM 对接适配器说明]] 进行深度集成。

*   产品对接范围：覆盖全部 C 端用户身份生命周期管理，包括用户注册（含手机号/邮箱/第三方账号注册）、实名认证（对接公安/运营商/银联等核验服务）、登录（账号密码、扫码、生物识别、无密登录）、会话管理、权限策略（基于角色/属性的细粒度访问控制）、账号自助服务（密码重置、绑定解绑、安全中心）、用户数据导出与合规删除（GDPR/PIPL 支持）；不包含员工身份（EIAM）、设备身份（DID）或物联网终端身份管理能力，相关需求请参考 [[EIAM 方案文档|EIAM 方案支撑文档]] 或 [[IoT 身份方案|IoT 身份接入指南]]。
